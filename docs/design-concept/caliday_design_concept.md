# CaliDay — Дизайн-концепция маскота и визуального стиля

**Версия:** 1.1
**Дата:** Февраль 2026

---

## 1. Маскот — Горо

### 1.1 Концепция

Горо — горилла-маскот приложения CaliDay. Центральный персонаж бренда.

**Почему горилла:**

- Гориллы используют калистенику в природе — лазают, висят, держат баланс на руках
- Сильная, но дружелюбная — мультяшный стиль делает её approachable
- Контрастно выделяется в сторе среди generic фитнес-иконок
- Отлично масштабируется: от 16px (силуэт читается) до полноразмерных анимаций
- Хорошо ложится на голубой бренд-цвет

**Имя:** Горо (от «горилла», коротко, запоминается)

### 1.2 Визуальные характеристики

- **Тело:** тёмно-серое с фиолетовым подтоном (#38384C основной, #424258 светлый)
- **Лицо:** тёплый бежевый/коричневый (#C4A882 → #A8896A)
- **Нос:** широкий, плоский, с выраженными ноздрями — характерная черта гориллы (#967756)
- **Глаза:** крупные, выразительные, белые с тёмными зрачками — дружелюбный мультяшный стиль
- **Сагиттальный гребень:** шишка на макушке — ключевой признак гориллы-самца
- **Надбровные дуги:** тяжёлые, массивные — создают характерный взгляд
- **Повязка на голове:** голубая (#A8D8FF) — бренд-акцент, спортивный элемент
- **Фигура:** треугольная — широкие плечи, мощные руки, узкий таз, маленькие ноги

### 1.3 Стиль рисовки

- Flat design с минимальными тенями
- Без контурных линий (формы создаются через разницу цветов)
- Крупные выразительные глаза
- Упрощённая анатомия с намёком на мускулатуру (грудные, бицепс, пресс)
- Масштабируемость: читаемый силуэт на любом размере

### 1.4 Позы

| Поза | Использование | Описание |
|------|---------------|----------|
| **Лицо** | Иконка приложения (основная), уведомления, аватар | Только голова Горо, занимает весь фрейм. Поддерживает смену выражений |
| **Флекс** | Главный экран, промо-материалы, стор | Одна рука на земле (горилья стойка), вторая флексит бицепс |
| **Idle** | Состояние покоя, меню, настройки | Стоит прямо, обе руки опущены, нейтральное дружелюбное выражение |

### 1.5 Система персонажей (перспектива)

| Персонаж | Роль | Характер | Визуал |
|----------|------|----------|--------|
| **Горо** (главный) | Наставник, тренер | Дружелюбный, подбадривающий | Горилла, голубая повязка |
| **Рэкс** | Мотиватор стриков | Энергичный, гиперактивный | Маленькая обезьянка, огненные элементы |
| **Бруно** | Демонстратор упражнений | Спокойный, техничный | Медведь, показывает технику |
| **Скала** | Челленджи и тесты | Строгий, но справедливый | Бык/буйвол, появляется на тестах |

## 2. Система выражений лица

### 2.1 Концепция

Иконка приложения — лицо Горо — является «живой»: выражение меняется в зависимости от состояния пользователя. Это создаёт эмоциональную связь с маскотом и дополнительно мотивирует пользователя через чувство ответственности перед персонажем.

Голова Горо занимает весь фрейм иконки без шеи и тела. Это позволяет менять только три зоны лица — брови, глаза и рот — сохраняя всё остальное (гребень, повязка, уши, нос, фон) неизменным.

### 2.2 Выражения

| Выражение | Триггер | Брови | Глаза | Рот |
|-----------|---------|-------|-------|-----|
| **Довольный** (default) | Тренировка выполнена сегодня | Спокойные, чуть приподняты | Открытые, зрачки по центру | Широкая улыбка |
| **Грустный** | Вечер, тренировка не выполнена | Домиком (внутренние концы вверх) | Зрачки смотрят вниз | Опущенные уголки |
| **Злой / мотивирующий** | Стрик под угрозой (за 2 часа до конца дня) | Сведены к центру, нахмурены | Сужены, зрачки прямо | Сжатый рот, решительный |
| **Спящий** | Ночное время (после полуночи) | Расслаблены | Закрытые (дуги вместо кругов) | Слегка приоткрыт |
| **Восторженный** | Достижение / новый ранг / рекорд | Высоко подняты | Широко открытые, зрачки-звёздочки | Открытый рот, широкая улыбка |
| **Поддерживающий** | Пользователь вернулся после пропуска | Мягко приподняты | Тёплый взгляд, зрачки чуть вверх | Лёгкая ободряющая улыбка |

### 2.3 Технические детали смены выражений

Иконка строится из слоёв. Неизменные слои:

- Фон (голубой градиент)
- Форма головы, шерсть
- Сагиттальный гребень
- Надбровные дуги (базовая форма)
- Повязка
- Уши
- Нос

Динамические слои (меняются в зависимости от выражения):

- Брови (path: угол наклона, кривизна)
- Глаза (ellipse: размер, форма, видимость зрачков)
- Зрачки (ellipse: положение cx/cy, возможная замена на звёздочки)
- Рот (path: кривизна, открытость)

### 2.4 Где используются выражения

| Контекст | Выражение |
|----------|-----------|
| Иконка на домашнем экране | Зависит от состояния (iOS Dynamic Icon / Android Adaptive) |
| Пуш-уведомление (утреннее) | Довольный |
| Пуш-уведомление (вечернее напоминание) | Грустный |
| Пуш-уведомление (угроза стрика) | Злой |
| Пуш-уведомление (поздравление) | Восторженный |
| Экран результатов (после тренировки) | Восторженный / Довольный |
| Главный экран (тренировка не начата) | Поддерживающий |
| Главный экран (тренировка завершена) | Довольный |
| Возвращение после пропуска | Поддерживающий |

## 4. Иконка приложения

### 4.1 Основная иконка — Лицо Горо

Основная иконка приложения — крупный план лица Горо, занимающий весь фрейм. Только голова, без шеи и тела. Этот формат выбран как основной по нескольким причинам:

- Лучше читается на маленьких размерах (48px, 32px) — лицо узнаваемо даже в миниатюре
- Поддерживает динамическую смену выражений — эмоциональная связь с пользователем
- Персонаж становится «живым» — реагирует на действия (или бездействие) пользователя
- Сильнее запоминается — лица привлекают внимание сильнее абстрактных символов

### 4.2 Дополнительные варианты

| Вариант | Файл | Использование |
|---------|------|---------------|
| **Лицо** (основная) | `caliday_icon_face.svg` | Иконка приложения, аватар, уведомления |
| **Флекс** | `caliday_icon.svg` | Стор-листинг, промо, главный экран |
| **Idle** | `caliday_icon_idle.svg` | Меню, настройки, состояние покоя |

### 4.3 Требования к файлам

- **Вектор-исходник:** SVG (для пересборки в любом размере)
- **icon.png:** 1024×1024 px, PNG, непрозрачный фон — flutter_launcher_icons нарежет все размеры
- **icon_foreground.png:** 1024×1024 px, PNG, прозрачный фон — для Android adaptive icon
- Скруглённые углы: `rx="224"` (22% от 1024) — iOS стиль

### 4.4 Масштабируемость

Иконка протестирована на размерах: 180px (iOS), 120px, 72px, 48px (Android), 32px, 16px. Силуэт читается на всех размерах.
